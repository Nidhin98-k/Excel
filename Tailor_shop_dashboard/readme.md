Created an interactive dashboard using a Tailor Shop dataset in Microsoft Excel. It contains KPI Cards, Pivot Charts, and Slicers to analyze orders, costs, garment types, and processing time.

KPI Cards: Total Orders, Total Cost, Average Cost per Order, Highest Cost Garment Type, Most Ordered Garment Type, and Average Processing Time.
Pivot Charts: Order Status Distribution, Cost by Garment Type, Orders by Garment Type, Monthly Cost Trend, Average Cost by Material, Average Cost by Garment Type, Monthly Order Trend, Material Distribution, and Average Processing Days by Garment Type.
Slicers: Order Status, Garment Type, Months, and Material.

The dashboard helps track tailor shop performance and provides valuable insights for better business decisions.
